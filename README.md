pets_lover
=====================

Organize sorted pets list by gender

Node and Yarn is required

### Run application

```
npm install yarn -g

yarn
npm start

```

```
****    Male
****       Garfield,Jim,Max,Tom
****    Female
****       Garfield,Simba,Tabby

```

### Run test

```
npm run test
```
